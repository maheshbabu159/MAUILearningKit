# MAUILearningKit

MAUILearningKit has the implementation to integrate api's developed in .NET ASP and show case the mobile development using .NET MAUI.

![Demo](demo.gif)


## Features

- Show customers
- Show orders
- Bar code scanner 
- Socket integration
